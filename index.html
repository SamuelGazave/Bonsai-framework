Bonsaï

A framework for e-mail design

Hi i'm Samuel. In my curent job i build a lot of e-mail campaigns and as html email can be tricky, i built a tool to help me create solid templates and newsletters very fast. It comes with a grid system and pre-built elements i use the most in my creations in order to help graphic designers to easily create mobile friendly designs.
